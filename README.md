# Job Application Tracker Portal

## Project Overview

Job Application Tracker Portal is a project designed to help students and job seekers organize their applications, monitor interview stages, analyze progress, and improve follow-up rates.

This project simulates a real-world application tracking system using Python and Google Colab while demonstrating CRUD operations, analytics, filtering, visualization, and keyword extraction.

## Features

* User registration simulation
* Add and manage job applications
* Track application stages
* Analytics dashboard
* Job description keyword extraction
* Search and filtering
* CSV export
* Visualization graphs

## Tech Stack

* Python
* Pandas
* Matplotlib
* Google Colab
* CSV Data Storage
